# Backlog and Bug Fixes

This document tracks planned features, improvements, and known bugs for Wealth Orbit. 

## 🚀 Planned Features (Backlog)
Contributions and Pull Requests are welcome for these items!

- **[ ] Add automatic savings addition (YoY/MoM)**: Schedule automatic, recurring additions to savings or investments (Year-on-Year or Month-on-Month) rather than manual one-off entries.
- **[ ] Improve piechart view**: Enhance the asset allocation breakdown with better interactivity, legends, and color palettes.
- **[ ] Add inflation adjustment**: Adjust projections and goals to account for historical or projected inflation rates.
- **[ ] Improve projection UI**: Enhance the layout, interactivity, and visual clarity of the projection charts and controls.
- **[ ] Add scenario comparison**: Allow users to create and compare multiple financial scenarios (e.g., "Aggressive Growth" vs. "Conservative FIRE").

### How to Contribute
1. Check the backlog above or the GitHub Issues tab.
2. Fork the repository.
3. Create a feature branch (ex. `git checkout -b feature/amazing-feature`).
4. Commit your changes (ex. `git commit -m 'Add some amazing feature'`).
5. Push to the branch (ex. `git push origin feature/amazing-feature`).
6. Open a Pull Request.
